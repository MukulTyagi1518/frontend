# Button Component

The Button component is a versatile and customizable button that can be used throughout your application.

## Basic Usage

<div className="flex justify-center p-4 w-auto h-auto bg-blue-600 border rounded-lg">
  <button className="px-4 py-2 font-bold text-white bg-blue-500 rounded-lg hover:bg-blue-600">
    Click me
  </button>
</div>

To use the Button component in your project, you can copy the following code:

```jsx
import React from "react";

const Button = ({
  children,
  onClick,
  variant = "primary",
  size = "medium",
}) => {
  const baseClasses = "font-bold rounded-lg transition-colors duration-200";
  const variantClasses = {
    primary: "bg-blue-500 hover:bg-blue-600 text-white",
    secondary: "bg-gray-200 hover:bg-gray-300 text-gray-800",
  };
  const sizeClasses = {
    small: "px-2 py-1 text-sm",
    medium: "px-4 py-2",
    large: "px-6 py-3 text-lg",
  };

  return (
    <button
      className={`${baseClasses} ${variantClasses[variant]} ${sizeClasses[size]}`}
      onClick={onClick}
    >
      {children}
    </button>
  );
};

export default Button;
```
